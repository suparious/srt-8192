# Contributing to Future Earth

Thank you for your interest in contributing to Future Earth! We're currently in the early stages of development, focusing on game design, documentation, and planning. Your input and ideas are valuable to us at this stage.

## How to Contribute

1. **Join the Discussion**:
   - Our primary platform for discussion is Discord. Join us at: <https://discord.gg/UMWQmCz7uF>
   - Introduce yourself and share your interests in the project.

2. **Review Existing Documentation**:
   - Familiarize yourself with the current game design documents in the `docs` folder.
   - Look for areas that need expansion, clarification, or improvement.

3. **Share Your Ideas**:
   - If you have ideas for game mechanics, features, or improvements, share them on Discord.
   - Be specific and explain how your ideas align with the game's overall concept.

4. **Contribute to Documentation**:
   - Help us expand and refine our documentation.
   - This can include game design documents, technical specifications, or user guides.

5. **Suggest External Resources**:
   - If you know of relevant articles, research, or tools that could benefit the project, let us know.

6. **Report Issues**:
   - If you find inconsistencies or problems in the existing documentation or plans, open an issue on GitHub.
   - Clearly describe the problem and suggest improvements if possible.

7. **Contribute Code (Coming Soon)**:
   - While we're not yet at the coding stage, we'll update this guide when we start accepting code contributions.

## Contribution Process

1. Fork the repository on GitHub.
2. Create a new branch for your contribution.
3. Make your changes or additions.
4. Submit a pull request with a clear description of your changes.

## Code of Conduct

Please note that this project is released with a Contributor Code of Conduct. By participating in this project you agree to abide by its terms. [Link to Code of Conduct]

## Questions?

If you have any questions about contributing, please ask on our Discord server. We're here to help!

Thank you for contributing to Future Earth. Together, we can create an amazing global leadership simulation game!
